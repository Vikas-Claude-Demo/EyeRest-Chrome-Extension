# Privacy Policy for EyeRest Chrome Extension

**Last Updated: April 15, 2026**

EyeRest is committed to protecting your privacy. This policy explains our "zero-data" approach to the EyeRest Chrome extension.

## 1. Zero Data Collection
EyeRest does **not** collect, store, transmit, or sell any personal data, browsing history, or identity information. We do not use any analytics or tracking services (like Google Analytics).

## 2. Permissions & Usage
EyeRest requests only the minimum permissions necessary for its core functionality:
- `alarms`: To run the vision timer accurately.
- `storage`: To save your settings (e.g., 20-min interval) locally.
- `notifications`: To send you break reminders.
- `tabs`: To open the exercise screen.

## 3. Local Storage
All your preferences are stored locally on your device using `chrome.storage.local`. No data is synced to any server or cloud service.

## 4. Open Source
The full source code of EyeRest is publicly available for review on [GitHub](https://github.com/drdhavaltrivedi/EyeRest-Chrome-Extension). This allows for complete transparency and verification of our privacy claims.

## 5. Contact
If you have questions, please reach out via our GitHub repository issues page.
