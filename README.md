# EyeRest: 20-20-20 Screen Timer

Protect your eyesight and eliminate digital eye strain with **EyeRest**, the most elegant and customizable break reminder extension for Google Chrome.

## Features
- **Custom Intervals & Durations:** Fully adjustable break intervals and durations.
- **Animated Exercise Screen:** A beautiful, auto-cycling guide of proven eye exercises.
- **Adaptive UI Design:** Seamlessly syncs with your system's Light or Dark Mode.
- **Custom Notifications:** Set dynamic desktop push notifications with personalized reminder messages.
- **Manifest V3:** Lightweight and secure.

## Installation
1. Download this repository as a ZIP or clone it.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the folder containing these files.
